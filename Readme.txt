***************Various Files*******************

1. image_rotation.py           ----> Project code

2. rotation_index.py           ----> Algorithm part

3. testDataCode.py             ----> Algorithm Testing code.

4. sort.py                     ----> Sorting the obtained Tested outputs to verify

5. haarcascade_frontal_alt.xml ----> file used to detect face using viola and jones algorithm

6. MI_research_paper.pdf       ----> Reference research paper

7. MI_first_review.pdf         ----> first review doc

8. MI_two_page.pdf             ----> 2 page research paper summarized doc

9. MI_presentation.pdf         ----> Project presentation

10. results.txt                ----> test image names along with rotation value in anti-clockwise direction. 

***************Various Folders*****************

1. testDataSet                 ----> Contains test images of particular image rotated with 1 degree of change between consecutive images.

2. Results                     ----> Output images generated with threshold of 1 degree.

3. results                     ----> Output images generated with threshold of 5 degrees.

4. documents                   ----> All the documents submitted like research paper, first review doc, research paper summary and presentation.

5. haarFiles                   ----> haar cascade files used in viola and jones algorithm.  

6. outputImage                 ----> output images with file name as anti clockwise rotation angle in degrees + inputfile name. 

7. outputValues                ----> contains results.txt

8. Rotations                   ----> contains rotations of input image
